重新定义 AI 幻灯片编辑

让 AI 生成的 HTML 演示稿可被人工修订、保存和导出——保真优先。

核心功能亮点

🎯

保真优先

不破坏 AI HTML 原有视觉效果和动画，修订层薄而透明。

✏️

修订模式

点击任意文本即可 inline 编辑，提交后立即生效。

💾

Patch 持久化

修改以 patch 形式存储，sourceHtml 永不被覆盖。

技术对比

方案保真度可编辑性导出质量
deck.json 重建低（有损转换）高中
sourceHtml + patches高（原始保留）中（渐进）高
纯 iframe 展示最高无原样

Patch 数据结构

每次用户修改生成一条 Patch，独立于 sourceHtml 存储，导出时重放应用。

type Patch = { id: string type: "text" target: PatchTarget value: string }
